## 📅 향후 구현 플랜 (Implementation Plan)

### 1. 전략의 핵심
`index.html`의 본문 내용을 갈아엎는 대신, **"이 사이트는 성격 테스트 모음집입니다"**라는 느낌을 주도록 **헤더(GNB)**와 **하단 추천 배너**를 추가하여 자연스럽게 확장을 시도합니다. (헤더 개선 완료)

### 2. Phase 2: 신규 테스트 추가 (Expansion)
*   **신규 테스트**: `love-test.html` (마라맛 연애 성향 테스트)
*   **컨셉**: "나의 연애 세포는 얼마나 타락(?)했나?"
*   **구조**: 기존 테토-에겐의 로직을 재사용하되 데이터만 교체하여 빠르게 확장.
*   **연결**: `index.html` 하단에 신규 테스트 유도 배너 추가.

### 3. 권장 작업 순서 (다음 단계)
1. `love-test.html` (마라맛 연애 성향 테스트) 신규 페이지 제작 및 `js/love-data.js` 구축.
2. 각 테스트 결과 페이지에 오픈그래프(OG) 이미지 동적 생성 로직 검토.
