## 📅 향후 구현 플랜 (Implementation Plan)

### 1. 전략의 핵심
`index.html`의 본문 내용을 갈아엎는 대신, **"이 사이트는 성격 테스트 모음집입니다"**라는 느낌을 주도록 **헤더(GNB)**와 **하단 추천 배너**를 추가하여 자연스럽게 확장을 시도합니다.

### 2. Phase 2: 신규 테스트 추가 (Expansion)
*   **신규 테스트**: `love-test.html` (마라맛 연애 성향 테스트)
*   **컨셉**: "나의 연애 세포는 얼마나 타락(?)했나?"
*   **구조**: 기존 테토-에겐의 로직을 재사용하되 데이터만 교체하여 빠르게 확장.
*   **연결**: `index.html` 하단에 신규 테스트 유도 배너 추가.

### 3. 권장 작업 순서 (다른 데스크탑에서 이어서 할 때)
1. `git checkout renewal-tetoegen`
2. `index.html` 시작 화면에 연령대 선택 `<select>` 또는 커스텀 드롭다운 추가.
3. `js/script.js`에서 선택된 연령 정보를 받아 결과 분석 시 활용하도록 로직 수정.
4. `js/teto-data.js`의 질문 내용을 더 자극적이고 재미있게 수정.
